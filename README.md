# Fleet StoryLab Portal

A trilingual GitHub Pages portfolio gateway for ten commercial vehicle product-planning, digital marketing, AI intelligence, safety, TCO, and education prototypes.

## Deployment
Upload all files to a GitHub Pages repository and open `index.html`. The hero animation is loaded from `video/index.mp4`; portfolio preview images are loaded from `images/`.

## Notes
This static site includes noindex meta tags, robots.txt, context-menu blocking, selection blocking, and obfuscated runtime data. These measures discourage casual copying and indexing, but client-side source code cannot be made technically unreadable after delivery to a browser.
